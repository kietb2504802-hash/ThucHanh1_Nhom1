## Mục tiêu
- Sử dụng AI để hỗ trợ cấu hình phần cứng/phần mềm (CLO13)
- Giải quyết sự cố kỹ thuật đơn giản với AI (CLO11)
- Sử dụng AI một cách đạo đức và minh bạch (CLO12)
- Rèn luyện kỹ năng tự học và khai thác công nghệ số (CLO14)

---

## Phần 1: Cấu hình phần cứng/phần mềm
- Sử dụng Google Bard/Gemini để tìm hiểu CPU, RAM, ổ cứng
- Kiểm tra bằng Task Manager / System Information
- Cài đặt và cấu hình Notepad++ để viết code Python
- File Python minh họa: hello_ai.py

Nguồn AI: Google Bard (Gemini)

---

## Phần 2: Giải quyết sự cố kỹ thuật
- Xử lý sự cố Wi-Fi trên Windows 10 bằng hướng dẫn AI
- Khắc phục lỗi Notepad++ không mở file Python lớn

Nguồn AI: Google Bard (Gemini)
